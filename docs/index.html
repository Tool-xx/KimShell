Что такое KimShell

🧬

Поддельный профиль железа

CPU, GPU, RAM, разрешение экрана и User-Agent генерируются случайно каждую сессию. Сайты видят несуществующий компьютер.

🧩

JS-расширение реального времени

Перехватывает Canvas, WebGL, Battery, AudioContext, Plugins и десятки других API до того как страница их прочитает.

🗂️

Изолированный профиль

Каждый запуск — новая временная директория. При закрытии все данные перезаписываются в 3 прохода и удаляются.

🛡️

Не трогает основной Brave

Завершение процессов строго по PID нашего экземпляра. Системный браузер, сессии и закладки пользователя не затрагиваются.

// getting started

Установка

  1. Клонировать репозиторий

    bash git clone https://github.com/Tool-xx/KimShell cd kimshell

  2. Установить зависимости

    bash pip install -r requirements.txt

    Требуется Python 3.11+. Зависимости: PyQt6, requests, psutil, pycryptodome.

  3. Запустить

    bash python main.py

    При первом запуске автоматически скачается Brave (~120 МБ). При повторных — загрузки нет.

  4. Опционально: закрыть DNS-утечку

    В Brave: Settings → Privacy → Use secure DNS → Cloudflare (1.1.1.1)

Вопросы и ответы

  • Brave скачивается каждый раз при запуске?
    Нет. Загрузка происходит только при первом запуске или если brave.exe удалён. Версия кешируется в файле .version внутри директории установки.
  • Закрывает ли KimShell мой основной Brave?
    Нет. Завершение процессов происходит строго по PID (taskkill /T /PID <наш_pid>), а не по имени процесса. Системный Brave, твои закладки и сессии не затрагиваются.
  • Где хранятся данные сессии?
    В %APPDATA%\KimShell\ — профили, карантин, расширение. Всё удаляется при закрытии если включена опция "Очистка при закрытии". Secure wipe: 3 прохода (0x00 → 0xFF → random).
  • Нужен ли VPN?
    KimShell скрывает аппаратный отпечаток браузера, но не IP-адрес. Для полной анонимности используй VPN вместе с KimShell.
  • Почему Canvas fingerprint всё равно уникален?
    Canvas noise делает хеш отличным от других пользователей — это и есть цель. Главное что он меняется каждую сессию и не совпадает с твоим реальным браузером.
  • Работает ли на Windows 7 / 8?
    Нет. Требуется Windows 10 1903+ для корректной работы PyQt6 и актуальных версий Brave.
